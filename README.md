# 🐺 Волчишка Bot v5.0

Кибернетический волчонок на Gemini AI для Telegram.

## Возможности

- 🧠 Умный чат на базе Gemini AI
- 🎤 Распознавание голосовых сообщений  
- 🔊 Озвучка ответов (edge-tts)
- 🎨 Генерация изображений
- 👤 Запоминание пользователей
- 🎙️ Плавная настройка голоса
- 🔄 Умная ротация API ключей
- 💾 Персистентное хранение данных

## Файлы для деплоя

```
volchishka-bot/
├── bot.py              # Основной код бота
├── requirements.txt    # Зависимости Python
├── Dockerfile          # Docker образ
├── .env               # Переменные окружения (создать!)
├── .gitignore         # Исключения Git
├── README.md          # Документация
└── data/              # Данные (создаётся автоматически)
    ├── users.json     # Данные пользователей
    ├── bot.log        # Логи
    └── histories/     # История сообщений
```

## Быстрый старт

1. Скопируй все файлы в папку проекта
2. Создай `.env` файл с токенами
3. Загрузи на bothost.ru
4. Запусти!
